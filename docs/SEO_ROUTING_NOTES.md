# SEO and Routing Notes for the GitHub Profile Portfolio

This repo is structured as a routing hub.
Instead of forcing every keyword and project description into one file, the README acts as the landing page and the `PAGES/` directory acts as the internal navigation layer.

## Why this helps

- more keyword coverage without making the homepage unreadable
- more internal links between related categories
- clearer paths for visitors coming from GitHub search, Google, or direct links
- easier future expansion as more repos are added
