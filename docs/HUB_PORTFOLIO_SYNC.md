# Portfolio ↔ TizWildin HUB Sync

Updated: 2026-07-07

## Purpose

The portfolio and the live TizWildin Entertainment HUB are intentionally different surfaces now. The portfolio proves Gary Doman's engineering identity and project range. The HUB routes products, public indexes, release proof, audio discovery, itch inventory, ARC source-spine links, and backend/licensing documentation.

## Fixed mismatch

The old portfolio described the HUB as only a desktop/plugin manager. The live HUB is broader: it is a public source-of-truth spine for free audio plugins, product routes, sample packs, game assets, release proof, JSON/LLM indexes, sitemap-backed crawl paths, ARC source routes, and documented API/licensing routes.
