# Sponsor Menu

## $1+ Micro-sponsor

Best for:

- one quick suggestion
- one small issue comment
- one README line review
- one plugin feedback note
- one repo first impression

## $5+ Small request

Best for:

- short checklist
- issue triage
- improved wording
- small docs improvement
- GitHub metadata suggestion

## $25+ Focused review

Best for:

- README review
- landing page notes
- plugin release feedback
- repo structure pass
- submission copy polish

## $100+ Deep review

Best for:

- full repo audit
- architecture review
- launch plan
- documentation rebuild plan
- CI/release strategy

## Custom

For implementation, long-form docs, plugin work, or multi-repo systems.

Sponsor amounts help prioritize time but do not buy guaranteed outcomes, acceptance, employment, press, or third-party approvals.
